This plugin allows to convert the text to an image, like this one has the option to use the web, even font that the visitor has not in your PC.

== Description ==

One problem applicant on the Web is the inability to use special plugin, with this plugin can work around this problem.
Indeed loading in the directory "fonts/" the font you want, we can convert any text string or title in the images.
The interface is very simple and with a very comprehensive guide that can be found at the following link.
http://text2image.webup.org

== Installation ==
1. Upload the directory text2image to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. You can find the plugin at menu Themes
4. Now you're ready to trasform the text in pictures :D
